Para las siguientes expresiones lambda:

Identificar las ocurrencias de variables libres y ligadas.
Reducir a su forma normal aplicando las reglas alfa, beta y eta, utilizando orden normal y orden aplicativo, y comparar los resultados.



-( λx.( ( λ y.y ) x ) ) z
Orden normal: primero el de mas a la izquierda
( λ y.y ) z
z
Orden aplicativo: primero el de adentro
( λx.x) z
z



-( λx.λy.x y ) ( z y )
Orden normal:
( λx.λw.x w ) ( z y )    alfa para que no se liga y
λw.z y w
zy

Orden aplicativo:
( λx.λw.x w ) ( z y )    alfa para que no se liga y
λw.z y w
zy


-( λx.λy.x ) x y
Orden normal:
((λx.λy.x) x) y
(λy.x) y --> recibe y pero en el cuerpo solo hay x
x

Orden aplicativo:
(λy.x) y
x



-(λx.((λz.z x) (λx.x))) y

(λz.z y) λx.x
(λx.x)y

Orden aplicativo:
(λx.(λx.x) x) y
(λx.x) y
y
